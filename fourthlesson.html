CSS-свойство Описание Примеры свойств
Padding Внутрений отступ элемента(от содержимого до внутренего края элемента) padding: 0 10px;(значения считываються по часовой стрелки начиная сверху); возможно установка для каждой стороны отдельно(padding-top/right/bottom/left)
Margin Наружный отступ от наружной границы до соседнего элемента или до границы родителя margin: 0 10px;(значения считываються по часовой стрелки начиная сверху); возможно установка для каждой стороны отдельно(margin-top/right/bottom/left; margin:(любое значение) auto - выравнивает элемент по центру по горизонтале)
Border-width Ширина обрамление border-width: любая еденица длины;
Border-style Оформление обрамления solid/dotted/dashed/double/inset/outset
Border-color Цвет обрамления Border-color: любой цвет
Display Тип элемента display: block/inline-block/inline
Verticle-align Вертикальное выравнивание Verticle-align: top/bottom/middle/baseline

Типы элементов

  1. Block - блочный элемент(занимает всю строку; можно установить ширину и высоту);
  2. Inline-block - строчно-блочный элемент(не занимает всю ширину строки, а только необходимое место, но в остальном соответствует блочному элементу;
  3. Inline - строчный(занимает только необходимое место, а не всю ширину родителя; нельзя задать width и height;
