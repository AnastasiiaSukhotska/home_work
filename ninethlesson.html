Псевдоклассы и псевдоэлементы

Псевдокдасс/псевдоэлемент Описание
:active срабатывает, когда на элемент нажтмают или используют кнопку Tab
:hover срабатывает когда на элемент наводят мышкой
:disabled находит любой отключенный элемент. элемент отключен, если не может быть активирован
:empty находит любой элемент, у которого нет потомков
:enabled находит любой включенный элемент. Элемент включен, если его можно активировать (например, выбрать, нажать на него или ввести текст) или поставить фокус
:first-child находит любой элемент, являющийся первым в своём родителе
:first-of-type находит первого потомка своего типа среди детей родителя
:focus применяется, когда элемент (такой как input формы) получает фокус. Обычно он активируется при клике мышью пользователем или при выборе элемента с использованием клавиши "tab" на клавиатуре
:invalid находит любые input или form элементы, контент которых не соответствует типу поля
:last-child находит любой элемент, являющийся последним в его родителе
:last-of-type находит последнего потомка с заданным тегом в списке детей
:not(x) функция, принимающая простой селектор X в качестве аргумента. Он находит элементы, не соответствующие селектору. X не должен содержать других отрицательных селекторов
:nth-child(an) находит один или более элементов, основываясь на их позиции среди группы соседних элементов
:nth-of-type() находит один или более элементов с заданным тегом, основываясь на их позиции среди группы соседних элементов
:optional находит любые input элементы, у которых не установлен атрибут required. Он позволяет формам легко отмечать необязательные поля, и давать им соответствующие стили
:read-only находит элементы, недоступные для редактирования пользователем.
:read-write находит элементы, доступные для редактирования пользователем, такие как текстовые поля
:required находит любые input элементы, имеющие атрибут required. Он позволяет формам легко сообщать, что поля должны быть корректно заполнены перед отправкой формы
::before применяется для отображения желаемого контента до содержимого элемента, к которому он добавляется. Работает совместно со свойством content
::after используется для вывода желаемого текста после содержимого элемента, к которому он добавляется. Pаботает совместно со свойством content.
::placeholder Псевдоэлемент, с помощью которого задаётся стилевое оформление подсказывающего текста, созданного атрибутом placeholder
