Команда Действие
npm i -g 'название плагина' команда для установки плагинов глобально(в систему, не в отдельный проэкт)
npm i -g gulp-cli установка командной строки gulp
npm init команда выполняеться один раз в папке проэкта для его инициализации. при ввеении команды консоль задает вопросы: имя проэктаб, версию, описание, точку входа, тестовую команду и ссылку на нид репозиторий и ключевые слова, автор, лицензию и все эти операции генерируют файл package.json и там их можно потом изменить
npm i plugin-name --save-dev установка плагина конкретно в проэкт и сохранения таким образом, что при закачке с git смогут подтянуться все установленые плагтны без их новой устаовки через команду npm instal, все установленные библиотеки сохраняються в файле package.json
npm i gulp --save-dev установка в проєкт самого gulp
создать файл в проэкте с именем gulpfile.js в этом файле пишем команды для gulp
'var имя переменной = require("имя модуля") в файле gulpfile.js создаем переменную для каждого модуля
gulp.task('task-name', (если задача кастомная-здесь необходимо задать финишь и указать эту же функцию в конце)=>{что нужно делать}) создание задачи для gulp
gulp.task('task-name', gulp.series("имя задачи1", "имя задачи2")) запуск выполнения серии других задач по очереди
gulp.task('task-name', gulp.parallel("имя задачи1", "имя задачи2")) задачи выполняються паралельно
return gulp.src("src/**/*.html")(пут окуда копировать)
.pipe(gulp.dest("dest/."))(куда копировать;dest-имя создаваемой папки(может быть любое))		 копирует файлы из одной папки в другую
npm i gulp-less --save-dev установка пакета less
gulp.task('less', ()=> {
return gulp.src('./less/**/*.less') .pipe(less()) .pipe(gulp.dest("dest/styles/.")); });		 задача для кампиляции файлоа less
gulp.watch("src/**/*.html", gulp.series(название задачи,которую надо запустить)) наблюдение за измнениями фйлов и при этих изменениях запуск необходимых задач
npm i browser-sync установка пакета для синхронизации с браузерами
gulp.task('browserSync', ()=> { browserSync.init({ server: { baseDir: 'dest/.' }, }) }) обновление браузера при обновлениях в документах
npm i --save-dev postcss для работы плагинов postcss
npm i --save-dev gulp-postcss для работы плагинов postcss
npm i --save-dev cssnano плагин для того что бы убрать пробелы и записывать код в одну строку для того что бы уменьшить вес выходного файла.
npm i --save-dev autoprefixer добавляет префиксы, что читалось более старыми версиями браузеров(для его работы необходимо добавить конфигурацию в package.json '"browserlist' "development")
npm i gulp-clean плагин для очистки папок
  1. в файл .gitignore добавляем папки node_modules, dest, package-lock.json
  2. для исользования cssnano можно добавить все его плагины в переменную
    • var postcssPlugins = { autoprefix(), cssnano() }
  3. затем добавить pipe(postcss(postcssPlugins)) к return
    4. затем можно подключить еще задать var rename = require("gulp-rename") и добавить pipe(rename({.min})) для добавления суфикса к минифицированому файлу

NB! Команда, которая выполняеться по умолчанию,должна носить имя default
