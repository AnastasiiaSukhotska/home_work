Подключение полей для заполнения

  1. Тэг form(парный)
  2. далее указываем атрибут action="куда отправлять информацию" и method="get(незащищенный,отображаеться в адресной строке) post(защищенный(пароли, логины))"
  3. внутрь этого тэга вкладываеться тэг input
  4. input имеет атрибуты type(тип вводимой информации), name(имя), value(значение)
  5. type может принимать значение text, number, range,tel, email,checkbox, color, submit(кнопка для отправки формы), reset(кнопка очистки формы)
  6. вместо input может быть select name, size(количество вариантов отображающихся одновременно)), multiple(возможость выбора больше одного варианта)
  7. можно вложить внутрь optigroup(для формирования груп выбора), option(вариант для выбора)
  8. вместо type может быть textarea для ввода многострочного текста, меняет свой размер в зависимости от количества строк, что запретить менять размер в css нужно задать resize:none
Вид атрабута type Описание Дополнительные атрибуты
text поле для ввода текста(фамилия, имя и т.д.) placeholder(подсказка, исчезает, когда начинают вводить текст), disabled(блокирует доступ к элементу), readonly(только для чтения), requared(обязателен к заполнению)
password для введения пароля теже
email для введения email(обязательно наличие @ при заполнении поля) теже
radio переключатель(если создать 2 с одинаковым name - выбрать можно только один disabled(блокирует доступ к элементу), readonly(только для чтения), requared(обязателен к заполнению)
checkbox флажок disabled(блокирует доступ к элементу), requared(обязателен к заполнению)
range диапазон disabled(блокирует доступ к элементу), max/min(максимальное и минимальное значение) step(шаг продвижения), requared(обязателен к заполнению)
